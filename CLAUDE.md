## Компоненты и UI

Сайт строится **без UI-библиотек** — всё пишется руками (inline styles + CSS-переменные).
Внешние зависимости только: GSAP (анимации), Next.js Image/Link (встроенные).

### Правило: перед любым новым UI-блоком

Перед созданием нового раздела или компонента — **покажи 2–3 варианта концепции**:
- Один абзац описания на каждый (макет, поведение, настроение)
- Жди выбора пользователя
- Только после выбора — пиши код

Примеры: карточки проекта (горизонталь vs сетка vs стек), CTA-секция (минималистичная vs акцентная), навигация (прозрачная vs залитая).

### Текущая структура страниц

| Страница | Ключевые секции |
|---|---|
| `/` (главная) | HeroSection, ServicesPreview, ProcessSection, TechSection, Testimonials, CTA |
| `/services` | Заголовок, список услуг (01–04), CTA |
| `/work` | Список проектов (карточки), фильтр по типу |
| `/about` | Intro-hero, основатель (фото + текст), философия (2 колонки) |
| `/contact` | Fullscreen CTA, Telegram + Max + Email |
